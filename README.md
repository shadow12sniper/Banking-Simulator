# Banking-Simulator

## Description
A multi-threaded banking simulator designed to handle concurrent financial transactions. This project demonstrates synchronization techniques to ensure thread safety during account operations.

## Key Functionalities
* Multi-threaded simulation of banking transactions.
* Implementation of thread-safe account balances.
* Demonstration of concurrency control in a financial context.

## Technical Stack
* C

## Setup
1. Clone the repository: `git clone https://github.com/shadow12sniper/Banking-Simulator.git`
2. Compile the source code: `gcc main.c -o bank_sim`
3. Run the application: `./bank_sim`
